Descrizione del sito

Pagina index.html

  • Header
    • Header creato con html con inserimento del immagine attraverso JavaScript (document.ready)
    • Utilizzo del display flex
    • Utilizzo del display table-cell per allineamento verticale
  • Main
    • La tabbella del main viene creato attraverso JavaScript (document.ready) tranne la prima riga
    • E' stato utilizzato BootStrap per la visualizzazione dei dati in formato tabbella
    • Ogni tupla è un'ancora che al suo click cambia intera pagina
      • Header cambia colore e l'immagine
      • La tabbella del main viene nascosta e visualizzata pannello BootStrap dei post con i suoi commenti
      • Footer cambia colore
    • pannello BootStrap descrizione
    • La lista dei post viene creata dopo aver cliccato su un'utente
    • I vari post vengono creati in sequenza e una volta terminato mostriamo la lista con un fadein()
    • in corrispondenza con ogni post sono presenti 2 bottoni uno per eliminare il post e l'altro per modificarlo
    • i due bottoni aprono 2 modali che richiedono input dall'utente per poter procedere
    • una volta confermate le modifiche eseguo una chiamata ajax dove invio le modifiche
  • Footer
    • Per la creazione del footer è stato utilizzato display flex
